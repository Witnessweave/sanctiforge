### \u2728 **2. Fixed Grid, Same Frame Size**
\u2705 Every cell should be the same size (e.g., 64\u00d764):
- Even if the sprite doesn\u2019t fill the cell, leave transparent padding.
- This lets you use \u201cregion\u201d or \u201catlas\u201d mode in your engine with no manual offsetting.

---

### \u2728 **3. Consistent Frame Order**
Within each row:
- Always start with the \u201crest\u201d frame.
- Progress through motion logically (frame 1 \u2192 frame 2 \u2192 frame 3 \u2192 frame 4).
- End with a looping frame if needed (e.g., frame 4 transitions back to frame 1).

`Walk_01 | Walk_02 | Walk_03 | Walk_04 | \u2026`

### \u2728 **4. Group Similar Orientations**
